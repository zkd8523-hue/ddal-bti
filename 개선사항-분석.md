## 🎯 1. 긴급 수정 필요 (High Priority)

### 1.1 이모지/동물 중복 문제
**문제**: 고양이가 2번 사용됨
- VSRN: 프로 효율러 고양이 😼
- FMAT: 세계관 창조주 고양이 🐈

**해결안**:
- FMAT를 다른 동물로 변경 추천
  - 예: 🦌 사슴 (우아하고 상상력 풍부한 이미지)
  - 예: 🦙 라마 (독특하고 예술적인 느낌)
  - 예: 🦚 공작 (화려하고 퍼포먼스적인 이미지)

### 1.2 이모지 오류
**문제**: VMAN 도파민 서퍼 미어캣 🦦 ← 이건 수달
**해결안**:
- 미어캣 이모지로 변경 또는
- 아예 "도파민 서퍼 수달"로 이름 변경 (수달이 더 귀여운 느낌)

### 1.3 명칭 어색함
**문제**: "방구석 무드디렉터 베어"
- "베어"는 외래어로 어색함
**해결안**: "방구석 무드디렉터 곰" 또는 "분위기 연출가 베어"

## 🔧 2. 개선 권장 (Medium Priority)

### 2.1 설명 문구 중복/유사성

#### VSAT vs FSAT (둘 다 Speed + Adventure + Tool)
**현재 문제점**:
- 둘 다 "빠르게", "실험적", "새로운 기구/장비"라는 비슷한 표현 사용

**개선안**:
- VSAT는 **"시각적 자극 + 속도"** 강조
  - "화면 속 장면을 보는 순간 바로 반응하는 빠른 실행력"
- FSAT는 **"상상력 + 기술 활용"** 강조
  - "상상을 현실로 구현하는 기계 활용의 달인"

#### VMRN vs FMRN (둘 다 Marathon + Routine + Natural)
**현재 문제점**:
- 둘 다 "이너피스", "순수함", "느리다" 등 유사 표현

**개선안**:
- VMRN은 **"시각적 완벽주의"** 강조
  - "같은 영상을 예술작품처럼 반복 감상"
- FMRN은 **"상상의 깊이"** 강조
  - "외부 자극 없이 순수 상상력만으로 깊은 몰입"

### 2.2 일관성 개선

#### 타이틀 패턴
현재 16개 중:
- 명사형: 11개 (예: "폭주기관차 토끼", "본능 저격수 여우")
- 설명형: 5개 (예: "침대 위 미식가 펭귄")

**제안**: 모두 "특징+동물" 형식으로 통일
- 예: "침대 위 미식가 펭귄" → "미식가 펭귄" (부제에서 설명)

#### 이모지 사용
- 각 설명의 첫 단어에 이모지가 불규칙하게 사용됨
- **제안**: 모두 제거하거나, 모두 일관되게 사용

## ✨ 3. 콘텐츠 품질 향상 (Low Priority)

### 3.1 특성 설명 강화

각 유형의 4가지 설명 구조를 패턴화:
1. **콘텐츠 선호**: 어떤 콘텐츠를 좋아하는가
2. **플레이 스타일**: 어떻게 즐기는가
3. **시간/속도**: 빠른가 느린가
4. **도구/방식**: 장비파인가 순정파인가

### 3.2 대표 키워드 추가

각 유형별 3개 핵심 키워드 추가 제안:
```typescript
keywords: ['빠른실행', '시각중독', '트렌드세터']
```

사용처:
- 공유 이미지에 표시
- 검색/필터링 기능
- 궁합 분석 시 활용

### 3.3 유형별 통계 정보

추가하면 좋을 데이터:
- 해당 유형의 예상 비율 (예: 15%)
- 다른 유형과의 궁합도
- 추천 콘텐츠 태그

## 🎨 4. 특정 유형별 수정안

### VSAT - 폭주기관차 토끼 🐰
**현재 Line 13**: "속전속결 실력파"
**수정**: "속전속결 실험파" (일관성: 뒤에서 실험가라고 표현)

### VSAN - 본능 저격수 여우 🦊
✅ 좋음. 수정 불필요

### VSRT - 명작 컬렉터 부엉이 🦉
**개선안**: "구관이 명관"이 다소 구식 표현
**제안**: "검증된 클래식"으로 변경 고려

### VSRN - 프로 효율러 고양이 😼
✅ 좋음. 수정 불필요

### VMAT - 방구석 무드디렉터 베어 🐻
**수정**: "베어" → "곰"
또는 타이틀 전체 변경: "분위기 연출가 베어"

### VMAN - 도파민 서퍼 미어캣 🦦
**수정**: 이모지 변경 (미어캣 아님) 또는 이름을 "수달"로 변경

### VMRT - 침대 위 미식가 펭귄 🐧
✅ 좋음. 수정 불필요

### VMRN - 사골국 순정 판다 🐼
✅ 독특함. 수정 불필요

### FSAT - 풀착장 우주 퍼그 🐶
**개선안**: "에베레스트 정상"이라는 비유가 갑자기 등장
**제안**: 더 자연스러운 표현으로 변경

### FSAN - 전두엽 풀악셀 치타 🐆
✅ 임팩트 있음. 수정 불필요

### FSRT - 템빨 마법사 댕댕이 🐕
✅ 재미있는 표현. 수정 불필요

### FSRN - 자연인 몽상가 나무늘보 🦥
✅ 컨셉 명확. 수정 불필요

### FMAT - 세계관 창조주 고양이 🐈
**수정 필요**: 고양이 중복
**제안**: 동물 변경 (🦌 사슴, 🦙 라마, 🦚 공작 등)

### FMAN - 무한리필 몽상가 햄스터 🐹
✅ 귀여운 컨셉. 수정 불필요

### FMRT - 망상 끝판왕 사자 🦁
**개선안**: "망상"이 다소 부정적일 수 있음
**제안**: "판타지 끝판왕 사자" 고려

### FMRN - 슬로우 모션 거북이 🐢
✅ 완벽함. 수정 불필요
